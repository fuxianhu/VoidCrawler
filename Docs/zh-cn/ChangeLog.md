# 更新日志

## v.0.0.1.1及以前

只是作者的一个小 demo，没有记录具体更新。至于Python版本（已经弃用）则曾经完全没有上传到Github。

那时Python版本已经有很多功能了，而C++的版本则只有打开cmd, powershell这样相当简单的功能。

## v.0.0.1.2

- 提供压缩包，解压后就是可执行文件和一些依赖库（DLL和一些必要文件）
- **船新视觉！相当好看！彩色动态渐变效果！**
- 增加一些基础的功能，包括但不限于：AudioPlayer, SwitchIME(Ctrl+Space,Shift,Ctrl)
- 暂不提供其他语言的翻译、安装程序、升级程序（这或许是百万年后的功能了吧 (:

> [!ATTENTION]
> 软件中的发布时间和实际发布时间完全不符，因为忘记在 Github 上发布了。
