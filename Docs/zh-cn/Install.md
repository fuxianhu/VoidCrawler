# 安装 Install

## Python 版本

> [!WARNING]
> Python 古老且今后不再受官方支持

Python 版本不提供可执行文件

前往 Python 官网（`https://www.python.org`）下载 Python 对应操作系统和架构的安装包并安装，正确配置 PATH 环境变量。

使用 pip 安装需要的依赖，并运行 `python \PythonVersion\main.py` 即可。

## 当前版本

在 `Releses` 页面中可能会提供 `Install.zip` 文件，解压。如果没有提供，到项目根目录下的 `Install` 目录找到并下载它们。

之后，运行 `Microsoft Visual C++` 安装程序（如果已经安装则可以直接退出），为了保证您的体验，还应为系统安装对应的字体文件（打开设置，找到`个性化`>`字体`）。

## 未来版本（画大饼

提供一个方便的 `VoidCrawlerInstaller.exe` 程序方便安装。
